# Emacs Find File

Emacs-style find-file functionality for VS Code and Cursor. Quickly navigate through files and folders with fuzzy filtering.

## Features

- Navigate directories like in Emacs
- Fuzzy search/filtering of files and folders
- Recent file history
- Quick traversal through directory structure
- Create new files by typing paths

## Usage

Run the command "Emacs: Find File" from the command palette (Ctrl+Shift+P / Cmd+Shift+P) or set up a keyboard shortcut.

## Requirements

None.

## Extension Settings

This extension does not add any VS Code settings.
